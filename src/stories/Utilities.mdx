<Meta title="Utilities/Guide" />

<Title>Utility Classes</Title>
<Subtitle>Quick reference for spacing, layout, typography, colors, and effects.</Subtitle>

<Description>
Use these classes anywhere you need lightweight styling without custom CSS.
All values are driven by CSS variables defined in the design system.
</Description>

## Spacing

```html
<div class="g-mt-4 g-mb-6 g-px-4 g-py-2">...</div>
```

## Layout

```html
<div class="g-flex g-items-center g-justify-between g-gap-4">...</div>
```

## Typography

```html
<h1 class="g-text-3xl g-font-bold">Heading</h1>
<p class="g-text-base g-text-muted">Body</p>
```

## Colors

```html
<div class="g-bg-primary g-text-light">Primary</div>
<div class="g-bg-success-light g-text-success">Success</div>
```

## Effects

```html
<button class="g-hover-lift g-press g-shadow">...</button>
```
